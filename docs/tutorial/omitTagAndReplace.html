Omiting a tag

Sometimes we want to omit the surrounding start and end tags but evaluating his contents. It can be done using data-tomit-tag attribute. If we want to do this unconditionally:

sample.html 
<html>
    <body>
        <div data-tdefine="user 'John'" data-tomit-tag="">
            <p>
                User: <span data-tcontent="user">the user</span>
            </p>
            <p>
                Message: <span data-tcontent="string: hello, ${user}!">hello, user!</span>
            </p>
        </div>
    </body>
</html>

The resulting HTML will be: 

<html>
    <body>
        <p>
            User: <span data-tcontent="user">John</span>
        </p>
        <p>
            Message: <span data-tcontent="string: hello, ${user}!">hello, John!</span>
        </p>
    </body>
</html>

If we want to do this depending on a condition:

sample.html 
<html>
    <body>
        <div data-tdefine="user 'John'" data-tomit-tag="omitDiv">
            <p>
                User: <span data-tcontent="user">the user</span>
            </p>
            <p>
                Message: <span data-tcontent="string: hello, ${user}!">hello, user!</span>
            </p>
        </div>
    </body>
</html>

The above example will omit the div tag if the variable omitDiv is true.

Replacing

The data-treplace attribute is similar to data-tcontent, but the first one removes the surrounding start and end tags:

sample.html 
<html>
  <body>
    <p>
      This is <span data-treplace="title">the title</span>
    </p>
  </body>
</html>

The resulting HTML will be: 

<html>
  <body>
    <p>
      This is hello world!
    </p>
  </body>
</html>
