ZPT-JS reference - Jquery expressions

  • Syntax.
  • Description.
  • Differences with ZPT.
  • Examples.

Syntax

jquery_expression ::= '$(' string ')'

Description

Jquery expressions evaluate arbitrary Jquery expressions.

Differences with ZPT

Jquery expressions does not exist in ZPT.

Examples

<div>
    <div>value1 = <span id="value1" class="values">10</span></div>
    <div>value2 = <span id="value2" class="values">20</span></div>
    <div>value3 = <span id="value3" class="values">30</span></div>
</div>

<div>
    <div>$('#value1') = <span data-content="$('#value1')">must be 10</span></div>
    <div>$('.values') = <span data-content="$('.values')">must be 10,20,30</span></div>
    <div>+: $('.values') = <span data-content="+: $('.values')">must be 10 + 20 + 30 = 60</span></div>
    <div>+: 100 $('.values') = <span data-content="+: 100 $('.values')">must be 100 + 10 + 20 + 30 = 160</span></div>
</div>
