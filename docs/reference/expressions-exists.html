Syntax

exists_expressions ::= 'exists:' expression

Description

Exists expressions test for the existence of paths. An exists expression returns true when the path expressions following it expression returns a value. It is false when the path expression cannot locate an object.

Note: some parts extracted from Zope Page Templates Reference.

Differences with ZPT

None.

Examples

Testing for the existence of a property: 

<div data-tcondition="not: exists: myObject/name">
    Name not set!
</div>
