ZPT-JS reference

Configuration options

  • command. The action to run. Available options are preload, partialRender and fullRender (default).
  • declaredRemotePageUrls. A list of URLs within external macros.
  • dictionary. An object containing key/values pairs accesible by the template.
  • i18n. An object to configure some setting about i18n.
  • maxFolderDictionaries. Sets the number of folder dictionaries to read.
  • root. The node(s) of the DOM to use as the root to process.
  • target. The node(s) of the DOM to use to process for partial rendering.
  • notRemoveGeneratedTags. Set whether ZPT-JS will remove the previously generated tags before processing.
